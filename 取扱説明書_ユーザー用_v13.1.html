はじめに

建築設備 知識クイズ Pro は、建築設備に関する幅広い知識を学べるクイズアプリです。全12ジャンル・2,911問の豊富な問題を収録し、3段階の難易度で学習できます。

主な特徴

  • 全12ジャンル・2,911問の豊富な問題
  • 3段階の難易度（Easy / Normal / Hard）
  • クラウド保存で複数端末からアクセス可能
  • 用語辞典（176語）・作業マニュアル搭載
  • ランキング機能で他ユーザーと競争

アカウント登録・ログイン

新規登録

  1. トップ画面の「新規登録」ボタンをタップ
  2. 漢字フルネーム（例：山田太郎）を入力
  3. パスワードを入力
  4. 「登録」ボタンをタップ

ログイン

  1. トップ画面の「ログイン」ボタンをタップ
  2. 登録済みの漢字フルネームを入力
  3. パスワードを入力
  4. 「ログイン」ボタンをタップ
パスワードの確認入力欄はありません。入力間違いにご注意ください。
データはクラウド（Firebase）に保存されるため、同じ名前・パスワードでログインすれば、別の端末からでも成績データにアクセスできます。ページをリロードしてもログイン状態は維持されます。

スタート画面

ログイン後のスタート画面では以下の操作ができます：

  • ジャンル選択（全ジャンルまたは個別ジャンル）
  • 難易度選択（Easy / Normal / Hard）
  • 出題数選択
  • クイズ開始
  • 用語辞典・作業マニュアルへのアクセス
  • 成績履歴の確認
  • ランキングの確認（画面下部）

クイズの始め方

  1. ジャンルを選択：「全ジャンル」または個別ジャンルのボタンをタップ
  2. 難易度を選択：Easy（★☆☆）/ Normal（★★☆）/ Hard（★★★）
  3. 出題数を選択：10問 / 20問 / 30問 / 50問 / 100問 / 全問
  4. 「クイズ開始」ボタンをタップ

クイズ画面の使い方

画面構成

  • 上部：進捗バー・問題番号・スコア
  • ジャンルバッジ・難易度表示（★マーク）
  • 問題文
  • 画像（ある場合）
  • 4つの選択肢（A / B / C / D）
  • ヒントボタン

回答方法

  1. 選択肢をタップして回答
  2. 正解は緑色、不正解は赤色で表示
  3. 解説が表示される
  4. 「次の問題」ボタンで次へ進む

ヒント機能

「💡 ヒントを見る」ボタンをタップすると、解答のヒントが表示されます。

途中操作

  • 「トップに戻る」：スタート画面に戻る（進捗リセット）
  • 「ここまでの結果を表示」：途中で結果を確認

結果画面

クイズ終了後、以下が表示されます：

  • 最終スコア（正解数 / 出題数）
  • 評価ランク
  • ジャンル別の正答率
  • 間違えた問題の確認

ランキング機能

スタート画面の下部に、難易度別（Easy / Normal / Hard）のランキングが表示されます。

  • 正解率の高い上位3名が表示されます
  • 全ユーザーの成績に基づいて算出されます

用語辞典・作業マニュアル

用語辞典

建築設備に関連する176語の専門用語を収録。読み仮名・解説付き。画像がある用語はタップで拡大表示できます。

作業マニュアル

現場での作業手順や注意ポイントをまとめたマニュアルです。

成績履歴

過去のクイズ結果を確認できます。

  • 日時・ジャンル・難易度・スコア・正答率が記録されます
  • クラウドに保存されるため、端末を変えても確認可能です

画像の拡大表示

問題に画像がある場合、画像をタップすると全画面で表示されます。

操作方法

操作動作
タップ画像を全画面で表示
ピンチ（2本指）拡大・縮小（1〜5倍）
ダブルタップ2.5倍に拡大 / 元に戻す
ドラッグ拡大時に画像を移動
マウスホイールPC使用時の拡大・縮小
×ボタン / 画像外タップ閉じる（非拡大時のみ）

アカウント管理（退会）

アカウントを削除（退会）することができます。

  1. 画面上部の「退会」ボタンをタップ
  2. 確認ダイアログで「OK」を選択
  3. アカウントと全ての成績データが削除されます
退会すると成績データは完全に削除され、復元できません。十分にご注意ください。

よくある質問

別の端末からでもデータを使えますか？
はい。同じ漢字フルネーム・パスワードでログインすれば、クラウドでデータが共有されます。
パスワードを忘れました。
管理者にご連絡ください。
ログインできません。
パスワードの大文字・小文字を確認してください。それでも解決しない場合は管理者にお問い合わせください。
成績がリセットされました。
同じ名前・パスワードでログインすれば成績は保持されます。別の名前でログインした場合、別アカウントとして扱われます。
オフラインでも使えますか？
初回はインターネット接続が必要です。データの保存・読み込みにもインターネット接続が必要です。
